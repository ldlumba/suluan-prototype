# Defense Demo Script

1. Open the Home Page and introduce SULUAN as a static prototype for institutional research intelligence.
2. Browse records and point out filters for department, year, research theme, access status, and validation status.
3. Search using Simulated Semantic Search and explain that results come from predefined temporary scenarios and fallback matching.
4. Open a Research Detail page from the result set.
5. Copy or view the generated citation and show related studies.
6. Explore the Knowledge Graph and explain paper, author, department, theme, keyword, and related-paper relationships.
7. Switch to the Admin role in the Role Demo page and describe simulated role-based access.
8. Submit or import metadata through the prototype-only metadata form.
9. Review the validation queue in the Admin Dashboard and simulate a local status change, not official institutional approval.
10. Show the Analytics Dashboard and clarify that charts are descriptive and based on fictional sample data.
