# Prototype Scope

SULUAN is implemented here as a static web prototype for proposal defense and user-flow validation. It demonstrates the intended experience of an AI-assisted institutional research intelligence platform without connecting to production systems.

The prototype simulates AI, database, authentication, and backend behavior. Semantic search uses predefined temporary scenarios and fallback keyword/theme matching. Role-based access, submitted records, and validation changes are stored only in the browser through localStorage.

No Supabase, Firebase, OpenAI API, external database, paid API, backend service, or confidential University of the Assumption record is used. All metadata is fictional sample content created for demonstration.

Full implementation, including real authentication, durable repository storage, administrative audit trails, production search, and approved AI-assisted features, will come after proposal approval.

The prototype demonstrates major user flows and intended system behavior: browsing research metadata, viewing records, simulating semantic discovery, exploring relationships through a graph, checking analytics, submitting metadata, switching roles, and reviewing a validation queue.
