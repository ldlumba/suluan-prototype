# Feature Checklist

## Implemented in prototype

- React + Vite + TypeScript project structure
- Tailwind CSS styling through the Vite plugin
- Professional SULUAN shell with collapsible navigation and text-based branding
- Home, Browse, Detail, Login, Create Account, Simulated Semantic Search, Knowledge Graph, Analytics, Admin, Submit / Import Metadata, and Role Demo routes
- Fictional sample research metadata with validated records separated from validation-workflow records
- Search and filter controls for validated normal browsing
- Citation display and copy action
- Recharts analytics
- React Flow knowledge graph
- localStorage for prototype-only session, account request, sidebar state, role, submission, and validation status changes

## Simulated in prototype

- Simulated semantic search over validated sample records
- Mock Login and Create Account interface flows
- Role-based access differences
- Metadata validation workflow
- Prototype-only metadata submission/import
- Administrative queue review
- Descriptive analytics generated from validated sample metadata by default
- Knowledge graph generated from validated sample metadata by default

## Planned for full system implementation

- Secure authentication and authorization after institutional approval
- Production database and repository storage after institutional approval
- Approved AI-assisted semantic search pipeline after validation
- Administrative audit logs
- Data privacy and retention controls
- Research office validation workflows
- Import tools for approved institutional records
- Advanced citation exports
- Accessibility and usability testing with target users

- Research-gap determination
- Originality checking
- Plagiarism checking
- Research quality scoring
- Academic merit judgment
- Predictive research planning
- Real authentication
